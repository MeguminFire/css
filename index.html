What is a Computer?

A computer is an electronic device that processes data and performs tasks according to a set of instructions, called programs. Computers are used for a variety of purposes, including communication, entertainment, work, and learning.

Main Parts of a Computer

  • CPU (Central Processing Unit): the brain of the computer.
  • RAM (Random Access Memory): short-term memory for quick data.
  • Storage: hard drives or SSDs for saving data long-term.
  • Motherboard: connects all components together.
  • Monitor, keyboard, and mouse: input and output devices.
